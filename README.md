# Aimos - AI Ambient Light

**"Connecting Mind and Space with Light & Music"**

Aimos is an AI-powered ambient lighting system designed to harmonize your indoor environment with the ever-changing sky of London. Developed by a Software Engineer with a background in Architecture and Music Composition.

## 🌟 Overview

Moving to London alone brought moments of solitude. Aimos was born from the desire to transform that solitude into comfort by synchronizing my personal space with real-time environmental data using Gemini AI.

## 🚀 Features

- **AI-Driven Atmosphere:** Analyzes real-time London weather and time via Gemini API to determine the optimal "Vibe".
- **Multi-Sensory Sync:** Synchronizes 24-bit RGB lighting (NeoPixel) with curated Spotify playlists.
- **Architectural Design:** Custom-designed 3D-printed facade inspired by architectural light diffusion.
- **Intuitive Web UI:** Control and monitor the synchronization through a sleek web interface.

## 🛠️ Tech Stack

- **Hardware:** Raspberry Pi Zero WH, NeoPixel LED Ring
- **Backend:** Python (Flask)
- **AI:** Google Gemini API
- **Music:** Spotify Web API
- **Design:** CAD, 3D Printing (PLA)
